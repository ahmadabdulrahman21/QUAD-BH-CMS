# QUAD Business House

**QUAD Business House** is a modern web platform designed to manage and showcase business content, services, and media in a structured and scalable way.

---

## 🚀 Features

- Dynamic content management system
- Admin dashboard for managing sections and media
- Image upload and storage support
- Responsive UI for all devices
- REST API integration with database
- Modular and reusable components (Next.js based)

---

## 🛠️ Tech Stack

- Next.js (App Router)
- React
- MySQL
- CSS / Tailwind CSS
- React Icons
